微软最近发布了一个跨平台代码编辑器Visual Studio Code(以下称VSCode)，使用了几天，感觉还不错。写篇博客记录一下。
我使用mac电脑，所以这里的按键都以mac为标准

1. 安装及配置

安装直接去官网下载，不解释。

在命令行调用VSCode

  • 设置

    1. 在.bash_profile中添加 

      code () {
    if [[ $# = 0 ]]
    then
        open -a "Visual Studio Code"
    else
        [[ $1 = /* ]] && F="$1" || F="$PWD/${1#./}"
        open -a "Visual Studio Code" --args "$F"
    fi
}

    2. 执行source .bash_profile
      参考

  • 调用

    • 使用code命令
    • code . 打开当前文件夹

配置文件

  • 在json配置文件 鼠标指向key 会自动弹出提示（可以所代表的功能）

2.文件操作

VSCode最多可同时并排显示三个文件。

  • 分栏编辑
    • cmd点击文件
    • cmd + \ 分两栏打开文件
    • 右键 -> Open to the Side
  • shift+cmd+P 可以打开所有命令
  • cmd+O 通过文件列表打开文件。
  • ctrl + tab 在已经打开的文件之间切换（类似cmd +tab切换应用）
  • ctrl + G 跳转到输入的行数
  • shift+cmd+O 打开一个文件的所有属性和方法列表。
    • 键入：可以分类浏览属性和方法
  • ⌘T通过一个名称打开一个方法或属性（可以跨文件）

  • ？ 可以打开特殊符号的列表

  • 代码格式化

    • 右键-> format Code

  • 括号匹配

    • ⌥⌘] 跳转到匹配的括号。

  • 多光标

    • 按住Alt点击，可以产生多个光标
    • ⌥⌘↓``⌥⌘↑ 向下和向下选择一行。
    • ⇧⌘L 选中文件中出现的某单词的所有位置。
  • 自动提示
    • editor.quickSuggestions editor.suggestOnTriggerCharacters 配置7x24 智能提示
  • 保存
    • 自动保存 ⇧⌘P, 输入auto 可以打开或关闭自动保存。

  • 搜索

    • ⇧⌘F 可全局搜索（支持正则表达式）
    • ⇧⌘J 打开或关闭高级搜索选项。
    • 右键 find in folder可以选择只在某文件夹搜索。

  • 转到定义

    • Ctrl 鼠标悬停预览，点击可以跳转。
    • Ctrl + Alt +点击 可以另外分出一栏打开定义。

  • peek
    *option + F12 可以通过预览窗口打开定义，点击标题文件名可以直接打开文件。

  • 被引用次数

    • editor.referenceInfos可以打开或关闭

  • 重命名 属性

    • 按下F2可以重命名。

  • 错误和警告

    • F8``⇧F8 循环显示一个文件中的错误和警告

视图

  • cmd + B 隐藏或显示侧边栏

版本控制

  • 文件变动指示器

    • 红色箭头表示被删除的行
    • 绿色竖线表示被添加的行
    • 蓝色竖线表示被修改的行

  • 文件比较

    1. 在左侧，右键文件select to compare，然后再右键一个文件Compare with ‘xx’，就可以出现diff。
    2. git的模式下，点击右上角的switch to changes file，就可以启动diff。

过滤多余的文件

  • 设置Setting
    • Windows %APPDATA%\Code\User\settings.json
    • Mac $HOME/Library/Application Support/Code/User/settings.json(官网提供的路径不对)
    • Linux $HOME/.config/Code/User/settings.json

常用快捷键

  • ⌘X 剪切行
  • ⌘C 复制行
  • ⇧⌘K 删除行
  • ⌘Enter 下面添加行
  • ⇧⌘Enter 上面添加行
  • ⌥↓ ⌥↑ 移动行
  • ⇧⌥↓ ⇧⌥↑ 复制行
  • cmd + 2 分割窗口打开文件
  • ⌥⌘↓``⌥⌘↑ 向上或向下一行添加光标
  • ⌥⌘] 跳转到匹配括号
  • ⌘] ⌘[ 添加或删除缩进
  • ⌘←``⌘→ 跳转到行首或行尾
  • ⌘/ 注释行
  • ⇧⌥A 注释块
  • ⌘F 发现
  • ⌥⌘F 更换
  • ⌘G 查找下一个
  • ⇧⌘G 查找上一个
  • ⌥⌘F 格式化代码
  • ⌘1``⌘2``⌘3 聚焦到左边中间或右边的编辑窗口
  • ⌥⌘← ⌥⌘→ 向左或向右切换编辑器
