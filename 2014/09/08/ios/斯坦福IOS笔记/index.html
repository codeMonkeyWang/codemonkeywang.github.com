IOS介绍

  1. MVC

  2. outlet 视图输出
  3. IOS7 可使用 @import Foundation; 导入foundation
  4. @property

    • @synthesize

  1. NSMutableArray
    • addObject: 添加对象到末尾
  2. 惰性实例化，在个体getter 中判断数组为nil，再实例化数组
  3. arc4Random() c函数库中获取整数
  4. NSUInteger 和unsigned int 类似，NSUInteger 在不同平台不同，iPhone5S下会是64位
  5. [@[“1”,”2”,”3”] containsObject:@”1”] 检测字符数是否包含在内。
  6. alloc 和init 方法要一起用，永远不要单独使用
  7. 实例类型 instancetype IOS7 新特性

  12. cmd+ = 图像调整为实际大小
  13. 调整图像按住option 可显示辅助线

oc

  1. CGPoint p =[obj getLocation];返回值为 struct ，需要检验没有将它发送给nil

  2. 动态绑定

    • 对象在运行时才确定要发送给什么类型的对象
    • NSString * s =@”x”;
    • id obj =s; //这完全合法，但是不安全
    • NSArray *a =obj; //完全合法，并且编译器不会报错
    • 绝对不要用id * ，id就是指针，这个指向指针的指针
    • code

  3. 内省

    • isKindOfClass //是否是某个类
    • isMemberOfClass //同上，不包括子类及其他类
    • respondsToSelector //是否对某一方法做出反应 @selector
    • [obj performSelector:shootAtSelector withObject:coordinate]; //方法要么没对象，要么有一个对象作为参数
    • [array makeObjectsPerformSelector:shootSelector]; //数组中所有对象执行该方法
  4. Foundation
    • NSArray
      • 将mutableArray 发送给copy 返回的是普通Array
      • mutableCopy 返回的是mutable的类型，不论是什么对象调用的
      • 希望对象可以被copy ，需要自己实现
      • 数组中所有对象都是强指针
      • sortedArrayUsingSelector:(SEL)selector; //根据SEL排序。
    • NSNumber
      • NSNumber *underline =@(NSUnderlineStyleSingle); //enum
      • NSNumber *match =@([card match:@[otherCard]]);
    • NSSet/NSMutableSet //对象唯一 ，高效，查找快
    • NSOrderedSet/NSMutableOrderedSet //有序集合
    • NSDictionary
      • 对象可以作为Key，但需要实现Hash和isequal.
    • 属性列表 集合中只有集合类（array dictionary NSNumber NSString NSDate NSData），成为属性列表
    • NSUserDefaults
      • setDouble: forKey:
      • setObject: forKey: 等等
      • synchronize 提交
    • NSRange
      • NSNotFound //返回范围为空（例如在数组中查找子字符串）
  5. UIKit
    • UIColor
      • 有alpha属性 //0-1
    • UIFont
      • preferredFontForTextStyle //首选字体
      • UIFontDescriptor //ios7 尝试将为一些字体添加粗体斜体等样式
    • NSAttributedString //不是String 的子类 ，为string添加描边，下划线等效果，该类是不可变的
      • string 方法
    • 属性字典
      @{ NSFontAttributeName : [UIFont preferredFontWithTextStyle:UIFontTextStyleHeadline] NSForegroundColorAttributeName :[UIColor blueColor],NSStrokeWidthAttributeName:@-5, //描边，负数表示填充并描边，正数描边空心 NSStrokeColorAttributeName:[UIColor orangeColor]} //描边颜色
    • UIButtong UITextView 的setArrributedTitle 设置属性字典
    • UILabel 有attributedText 属性 是只读的，只能创建可变副本，在设置回去
    • UITextView
      • TextStorage; //NSMutableString 的子类
        • [self.body.textStorage addAttribute:NSForegroundAttributeName value:sender.backgroundColor range:self.body.selectedRange] //设置选中的内容为触发按钮的背景色
        • [self.body.textStorage addAttributes:@{NSStrokeWidthAttributedNmae:@-3 NSStrokeColorAttributeName:[UIColor redColor] range:self.body.selectedRange]; //设置描边
        • [self.body.textStroke removeAttribute:NSStrokeWidthAttributName range:self.body.selectedRange];
      • 可以创建一个限制区域，使文本环绕图片显示（TextKit）
    • Attributor 属性化字符串
      • selectable //可选择的
      • editable //可编辑的
  6. ViewController
    • ViewDidLoad //适合init 只会调用一次
      • 不要添加任何视觉图形的初始化设置（这时图形未加载）
    • viewWillAppear: //view将要显示时调用，
      • 不要在这里添加一次性初始化代码，多个mvc存在时可能会被多次调用。
      • 如果视图切到后台时一些数据发生变化，可以在这里重新加载
    • viewWillDisappear: //view将要退出显示，在这里释放资源
    • viewDidAppear: & viewDidDisappear: //view 已经完成某些操作时调用
    • view{Will,Did}LayoutSubviews: //在屏幕横竖屏切换时，重新布局子控件时调用
    • Autorotation //自动旋转
    • didReceiveMemoryWarning //内存不足
    • awakeFromNib //解析nib storyboard
    • 调用顺序
      • Instantiated
      • awakeFromNib
      • Outlets get set
      • viewDidLoad
      • ViewWillLayoutSubviews:
      • viewWillAppear: adn viewdidAppear:
      • viewWillLayoutSubviews: and viewDidLayoutSubviews:
      • didReceiveMemoryWarning
  7. Notifications //广播
    • NSNotificationCenter
      • [NSNotificationCenter defaultCenter] //返回单例
      • addObserver:(id) observer //想要接收对象，一般为self
        selector:(SEL) //执行的方法
        name:(NSString) //广播名称
        object:()sender; //表示只对某个对象发出的广播感兴趣
      • 方法接收notification对象
        • notification.name,notification.object,notification.userInfo
      • [center removeObserver:self] [center removeObserver:self name:UIContentSizeCategoryDidChangeNotification object:nil]
        • 一定要移除广播，广播站会保留一个不安全保留 类型的指针,如果没有调用上边方法就离开堆，当系统向你发送广播时，你会崩溃。（ios6 以上改为了weak）
      • Unsafe retained //不安全保留,如果没有调用就离开了堆，通知中心会试着给你发送一个通知，程序会崩溃
      • 在MVC离开屏幕是，就应该移除广播 可以在dealloc中移除（其它情况不建议使用dealloc）
      • UIContentSizeCategoryDidChangeNotification() //字体设置改变的广播

      视图、绘制、手势识别

  • Views
    • addSubview:(UIView *)aView
    • removeFromSuperview;
    • self.view 是顶级视图
    • 设置UIView 为透明 //性能开销很大
      • opaque = NO //透明
      • alpha
    • 实现 方法 drawRect:(CGRect) aRect;//不要调用，留给系统调用的
      • 使用- setNeedDisplay;
      • 系统使用Core Graphics实现
    • UIBezierPath 绘制图形
    • CGContextRef context = UIGraphicsGetCurrentContext(); //获取系统上下文，是个无类型的指针
  • UIGestureRecognizer //手势识别
    • UIPanGestureRecognizer
      • (CGPoint)translationInView:(UIView *) aView; //触摸移动距离
      • (CGPoint)velocityInView:(UIView *) aView; //移动速度
      • (void)setTranslation:(CGPoint)translation inView:(UIView **)aView //对移动距离重置
      • 要注意Failed 和 Canceled 的手势状态

  • block
    • __weak MyClass *weakSelf = self //block中引用此指针，解决强引用环
