UIScrollView

  1. scrollView.ContentSize; scrollview的大小
  2. showsHorizontalScrollIndicator =NO 隐藏水平滚动条。
  3. scrollView.bounces =NO; //去掉弹簧效果
  4. scrollView.contentInset =UIEdgeInsetsMake();// 上左下右添加额外的滚动区域
  5. scrollView.contentOffset //现在显示的内容
  6. maximumZoomScale =2.0 //最大放大倍数
  7. minimumZoomScale = 0.2 //最小比例
  8. delegate //设置代理，需要实现协议
    • ViewForZoomingScrollview:(UiScrollView * )scrollView //缩放
    • scrollViewDidScroll:(UIScrollView * ) scrollView //滑动时调用
    • scrollViewDidEndZooming: //缩放结束
    • scrollViewDidZoom //缩放时调用
  9. pagingEnabled =YES; //是否分页

动画

  1. [ UIView beginAnimations:nil context:nil]; 一般参数填nil
  2. [ UIView commitAnimations] 提交动画，和beginAnimations中间的代码使用的动画。
  3. [ UIView setAnimationDuration:2.0] 动画时间
  4. CGAffineTransformMakeRotation(- M_PI_4); CGAffineTransformRotation(sender.transform)左旋转45度
  5. CGAffineTransformMakeScale(); 放大缩小
  6. 给Transform赋值CGAffineTransformIdentity 恢复默认值
  7. 帧动画
    • image.animationImages =@[@"test1.jpg",@"test2.jpg"] //动画数组
    • [image startAnimating]; //开始播放动画
    • image.animationDuration =0.1*count //持续时间
    • image.animationRepeatCount =1; //播放次数
    • isAnimation //是否正在播放
  8. UIView animateWithDuration: animations:^{…}]; //执行动画
  9. UIView animateWithDuration: animations: completion: //动画完成调用某方法

    void (^myBlock)();

  2. 如何利用block封装代码

  3. block访问外面变量
