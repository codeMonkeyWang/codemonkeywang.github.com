基础

基本技巧

控件的位置调节技巧

  • 屏幕的高度默认转换为3d坐标2个单位
  • widget的anchor 如果Left和Right同时勾选，会压缩UI的宽度

root节点

  • 创建 Tools -Daikon.. -UI wizard
  • 需要先设置Atlas和font。
  • Orthographic: 如果勾选了，表示是基于2D的，否则是基于3D的。可以更改此选项，做出效果的UI
  • Pixel Perfect:如果勾选了，表示是按照图片原尺寸显示，没有任何拉伸或缩放，否则容易产生变形和模糊。
  • User Joystick:是否支持手柄等外设。
  • Joystick Click Button:设置与鼠标左键点击效果一样的手柄按键。

右键功能

  • 添加组件
  • 可使控件填满父类

公共属性

  • anchor
    • 勾选 proportional 成比例的变化。
    • 可多选，
    • 同时选 Left和right 缩放时会挤压Label的大小
  • Interactive 是否可交互
  • can Focus 可以获取焦点
  • Clip Children 使子节点不超过父节点大小

Label

  • 选中Scene 中UIRoot 点击右键-Add control-Label
  • 属性：
    • Auto Size 自适应大小
    • Auto height 自适应高度
    • Text Slign 左右对齐方式
    • Vert Align 上下对齐方式
    • Word wrap 自动换行
    • process markup label可采用多种颜色 [color #ff0000] [/color]
    • 可以添加sprite [sprite “grid”]
    • 可以识别\n 换行符
  • Process Markup 和Colorize Sprites 没明白

Sprites

公共属性

  • Filp Horz 水平镜像
  • Filp Vert 垂直镜像

  • fill 填充方向 及填充百分比

  • Basic

  • Sliced
    • 双击sprite 可以进行编辑
  • Radial
  • Tiled
  • Texture
  • Animated
    • 在Tools菜单可以创建 Animation Clip。
  • web

Button

  • Button上点击右键可以创建脚本。

Atlas

  • 选中几个图片-右键-Daikon forge -texture Atlas-creat New Atlas

Font

  • 动态字体
  • 静态字体
    • 使用bitmap font（bmfont）

高级控件

checkbox

  • check Icon
  • Label
  • Group 所在的分组，可以为任何df 控件

Dropdown List 下拉列表

  • trigger 用来触发的df组件
  • 使用Selected　Index 获取选项

ListBox

  • 跟上边差不错

Progress Bar

  • Fill Mode Fill 填充 stretch 拉伸
  • Act as Slider 是否可以交互

richtextLabel 富文本

  • Auto scale Control根据控件大小缩放，Screen 根据屏幕缩放。
  • Keep Whitespace 是否保持文本中的空格。
  • Scrolling 滚动条相关属性

Slider

  • track 滑动器背景图片
  • Step 每次步进的精确度
  • Scroll Size ?????

textBox

  • 无法输入中文
  • Blank 闪烁光标

ScrollBar

  • track 背景
  • thumb 图标
  • Auto Hide 自动隐藏
  • Increment 每次移动的值

Scroll Panel

  • Auto Layout 自动布局
  • Wrap Layout 填满一行在填下一行

tab Control

  • tab Container 和tabstrip 一起使用
  • 在tabstrip 中关联 tabControl
  • tab strip 可以通过右键添加tab
  • 在编辑界面双击tab可以切换到对应的page

Drag Handle

  • 在控件底下添加此控件为子控件，即可实现拖动

Resize Handle

  • 在控件底下添加此控件为子控件，即可实现拉伸控件大小

Data Binding

property Binding 属性绑定

  • Data Source 为控制的控件 （为GameObject类型，可关联任何public属性值）
  • Data Target 为变化组件的属性
  • format 中需要填的东西为String.Format（）具体使用方法可参考官方文档:
    http://msdn.microsoft.com/zh-cn/library/dwhawy9k(v=vs.110).aspx

  • function 为要显示的特效
