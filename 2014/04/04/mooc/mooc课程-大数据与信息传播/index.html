第一课

  • 计算机信息技术的发展，造就了数字媒体，改变了人、媒体、信息之间的关系。
  • 主要研究对象： 人、媒体、信息之间的关系
  • 三个概念：大数据、社会化媒体、信息传播
  • 三个V
    • Volume,数据量大
    • Variety ，数据多样化
    • Velocity 爆炸的方式增长
  • 互联网改变了人和人的聚集方式
  • 话题
    • 我们是否认清并理解了今天的互联网世界的基本结构
    • 我们是否能识别并洞察生存在社会化媒体中的各类人群？
    • 我们是否能理清网络社会人群的镶嵌、连接和涌现的规律。
    • 我们是否能过获知网络人群“互动、对话”的缘由
    • 口碑的产生、作用及社会影响。
    • app
    • 网络公关和网络营销
    • 搜索引擎
  • 150定律
    -每个人有30个强关系和120个弱关系

第二课 探究互联网世界的基本结构 –0409

  • 媒体在互联网的重要任务：了解和掌握信息获取者的注意力
  • 互联网是一个动态的、立体的、无边界的信息世界。

    本章核心概念

  1. 大数据与数据挖掘
  2. 互联网世界
  3. 信息消费
  4. 信息传播路径

问题

  • 网络信息传授双方的博弈：如何捕捉信息获取者的注意力。
  • 面临社会化媒体，媒体到达问题。

大数据挖掘

  • 数据量大
  • 类型多样
  • 爆发式增长
  • 意见相同者聚集而成的”信息茧房”，即使私人空间又是公共空间。

互联网的基本结构

  • 如何评估全媒体发行的效果
  • AISAS模型

社会化媒体中的人群识别 –0415

  • 社会化媒体已从大规模轰炸转到精准投放
  • 改变了人在信传播中的地位、作用和权利
  • 社会化媒体上的人群隐匿性。

社会化媒体中的人群识别 方向

  • 人群识别
  • 用户行为数据
  • 用户生成内容数据
  • 精准营销
  • 实时竞价
  • 需求方平台
  • 数据管理平台

人群识别

  • 语音识别
  • 人脸识别
  • 通常采用cookies
